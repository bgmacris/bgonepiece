Algunos de mis proyectos

Algoritmo A* Snake Game Repositorio: GitHub Tecnologias: Python Algoritmo de hace 3 años, inicios de estudios en IA. Actualmente estoy reformando todo el algormitmo, para poder optimizar el codigo y que sea mas eficiente. El codigo utiliza PyGame para la interfaz y OOP para implementar una busqueda informada, utilizando el algoritmo A*, el codigo tambien incluye una implementacion el algoritmo dijkstra.
Algoritmo minimax TicTacToe Game Repositorio: GitHub Tecnologias: Python El codigo utiliza PyGame para la interfaz y OOP para el algoritmo. El algoritmo minimax recorre el árbol de juego en profundidad hasta alcanzar un movimiento. Cada movimiento posible es evaluado segun la funcion de evaluación definido. Es una practica sencilla, y en este caso es viable poder recorrer y calcular todos los nodos, no supone una gran capacidad de calculo.
One Piece scraping Tecnologias: Python Aplicacion creada para la comodidad de leer el manga de one piece en un dispositibo ebook. Automatizando la descarga y exportacion de tomos de una web. Es una simple aplicacion que esta formada por un crawler que descargar tomos del manga y otro script que se encarga de crear los PDFs de los tomos extraidos. Proximamente me gustaria implementar una API para la web, siendo esta aplicacion una de sus diferentes funciones.
